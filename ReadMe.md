Git的一些使用：

    `git init` 是 Git 版本控制系统的一个基础命令，它的主要作用是 **在当前目录初始化一个新的 Git 仓库** 。

这是任何 Git 项目的起点，为后续的代码管理和版本控制奠定基础。

    `git commit -m` 是 Git 版本控制系统中用于**提交更改并附带提交信息**的命令。

```
git commit -m "提交信息"
```

这个命令会创建一个新的提交记录，包含：

* 你之前用 `git add` 添加到暂存区的所有文件更改
* 你在 `-m` 后面提供的提交信息
* 提交时间和作者信息示例
